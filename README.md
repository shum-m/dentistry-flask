# dentistry-flask
Приложение автоматизирующее деятельность стоматологической клиники.

# О приложении
1. python 3.7
2. postgresql
3. flask
4. flask-sqlalchemy
5. flask-login
6. flask-bootstrap
7. flask-wtf
8. flask-migrate
